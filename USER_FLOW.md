## 1. 전체 플로우 개요

```
[시작]
  │
  ├─ 관리자 ──────────────────────────────────────────────────────────────┐
  │    │                                                                  │
  │    ▼                                                                  │
  │  로그인 → 홈 대시보드                                                  │
  │    │                                                                  │
  │    ├→ ① 직원 관리 ─────────── 등록 / 수정 / 비활성화                    │
  │    │                                                                  │
  │    ├→ ② 4주 기간 설정 ──────── 시작일 선택 → 매장 휴무일 지정            │
  │    │                                                                  │
  │    ├→ ③ 고정 휴무 승인 ─────── 신청 목록 확인 → 승인/거절               │
  │    │                                                                  │
  │    ├→ ④ 스케줄 생성 ────────── 자동 생성(7단계) → 결과 확인              │
  │    │         │                                                        │
  │    │         ├→ 수동 수정 ──── 셀 편집 + 실시간 검증                    │
  │    │         ├→ 재생성 ─────── 랜덤 재시도                              │
  │    │         └→ 확정 ────────── 잠금 + 직원 공유                        │
  │    │                                                                  │
  │    ├→ ⑤ 내보내기 ──────────── 엑셀 / PDF 다운로드                      │
  │    │                                                                  │
  │    └→ ⑥ 이력 / 템플릿 ─────── 과거 조회 / 패턴 재사용                   │
  │                                                                       │
  ├─ 직원 ────────────────────────────────────────────────────────────────┐
  │    │                                                                  │
  │    ▼                                                                  │
  │  로그인 (PIN/링크) → 홈                                                │
  │    │                                                                  │
  │    ├→ ① 고정 휴무 신청 ─────── 캘린더에서 날짜 선택 → 신청              │
  │    │                                                                  │
  │    └→ ② 내 스케줄 조회 ─────── 4주 캘린더 뷰 + 금토일 현황              │
  │                                                                       │
  └───────────────────────────────────────────────────────────────────────┘
```

## 2. 관리자 플로우

### 2.1 로그인 및 홈

```
[앱 접속]
  │
  ▼
[로그인 화면]
  │  - 관리자 계정으로 로그인 (ID/PW 또는 NextAuth)
  │
  ▼
[관리자 홈 대시보드]
  │  - 현재 4주 주기 요약 카드
  │  - 진행 상태 표시: 기간 설정 → 고정 휴무 → 스케줄 생성 → 확정
  │  - 빠른 액션 버튼
  │
  ├→ [직원 관리]
  ├→ [4주 기간 설정]
  ├→ [고정 휴무 승인]
  ├→ [스케줄 생성/조회]
  ├→ [내보내기]
  └→ [이력/템플릿]
```

### 2.2 직원 관리 (F-01)

#### 직원 등록 플로우

```
[직원 관리 화면]
  │
  ├→ [+ 직원 등록] 버튼 클릭
  │     │
  │     ▼
  │   [등록 폼]
  │     │  - 이름 입력 (필수, 한글 최대 20자)
  │     │  - 직급 선택 (필수): 점장 / 부점장 / 일반직원
  │     │  - 고용 형태: 정규직 / 파트타임
  │     │  - 연락처, 입사일, 메모 (선택)
  │     │
  │     ▼
  │   {이름 중복 확인}
  │     │
  │     ├─ 중복 없음 → [저장] → 직원 목록에 반영 ✅
  │     │
  │     └─ 중복 있음 → [경고 팝업] '동일 이름의 직원이 있습니다'
  │                      │
  │                      ├─ [계속 등록] → 저장 ✅
  │                      └─ [취소] → 폼으로 복귀
  │
  ├→ [직원 행 클릭] → 수정 모드
  │     │  - 모든 필드 수정 가능
  │     │  - 직급 변경 시: 다음 4주 주기부터 새 규칙 적용
  │     │  - 변경 이력 자동 저장
  │     └→ [저장] → 목록 갱신 ✅
  │
  └→ [비활성화] 버튼
        │
        ▼
      {확정 스케줄에 배정 여부 확인}
        │
        ├─ 배정 있음 → [경고] '확정 스케줄에 배정된 직원입니다'
        │                 │
        │                 ├─ [계속] → 비활성화 (목록에 회색 표시)
        │                 └─ [취소]
        │
        └─ 배정 없음 → 즉시 비활성화 ✅
```

#### 직원 목록 조회

```
[직원 목록]
  │  - 기본 정렬: 직급순 (점장→부점장→일반) → 이름 가나다순
  │  - 필터: 직급, 활성/비활성
  │  - 각 행: 이름, 직급, 고용형태, 최근 활동 4주 주기
  │
  └→ 행 클릭 → 상세/수정 화면
```

### 2.3 4주 기간 설정 (F-02)

```
[4주 기간 설정 화면]
  │
  ▼
[기간 선택]
  │  - 방식 ①: 시작 날짜 직접 선택 (Date Picker)
  │  - 방식 ②: 연도 + ISO 주차 번호 입력
  │
  ▼
[종료일 자동 계산] → 시작일 + 27일 = 종료일 (28일 고정, 수정 불가)
  │
  ▼
{동일 기간 스케줄 존재 확인}
  │
  ├─ 존재함 → [경고] '해당 기간의 스케줄이 이미 있습니다. 덮어쓰시겠습니까?'
  │              │
  │              ├─ [덮어쓰기] → 기존 초기화 후 진행
  │              └─ [취소]
  │
  └─ 존재하지 않음 → 진행
       │
       ▼
[28일 캘린더 렌더링]
  │  - 4주 × 7일 그리드 표시
  │  - 날짜 헤더: 요일 + 주차 표시
  │
  ▼
[매장 지정 휴무일 지정]
  │  - 날짜 클릭 → 파란 배지 토글 (매장 지정 휴무일 ON/OFF)
  │  - 재클릭 → 해제
  │  - 동일 주 여러 날짜 지정 가능
  │
  ▼
[주차 요약 패널] (실시간 갱신)
  │  - 각 주차별: 날짜 범위, 유형(주3회/일반주), 지정 날짜 수
  │  - 합계: 주3회 N주 / 일반 M주
  │
  ▼
[설정 저장] 버튼
  │
  ├─ 저장 성공 → '스케줄 생성' 버튼 활성화 ✅
  │
  └─ 매장 지정 휴무일 없이 저장 → 허용 (4주 전체 일반주 처리)
```

```
  │  - 체크박스로 다수 선택
  │  - [일괄 승인] / [일괄 거절] 버튼
  │  - 과밀 날짜 포함 시 경고 후 진행
```

### 2.5 자동 스케줄 생성 (F-04/05/06)

```
[스케줄 생성 화면]
  │
  ▼
{사전 조건 확인}
  │
  ├─ 직원 0명 → ❌ '등록된 직원이 없습니다. 먼저 직원을 등록해 주세요.'
  ├─ 기간 미설정 → ❌ '4주 기간을 먼저 설정해 주세요.'
  └─ 조건 충족 → 진행
       │
       ▼
     [🔄 자동 생성] 버튼 클릭
       │
       ▼
     ┌──────────────────────────────────────────────────┐
     │         7단계 자동 생성 알고리즘 실행              │
     │                                                  │
     │  STEP 1. 고정 휴무 우선 배정                      │
     │    └→ 승인된 고정 휴무 → 해당 날짜에 배정          │
     │    └→ 주별 잔여 횟수 차감                         │
     │    └→ ⚠️ 주별 한도 초과 시 경고                   │
     │                                                  │
     │  STEP 2. 점장/부점장 매장 휴무일 제외              │
     │    └→ 매장 지정 휴무일에 출근 강제 고정            │
     │    └→ ❌ 고정 휴무 충돌 시 생성 중단               │
     │                                                  │
     │  STEP 3. 주차별 잔여 휴무 횟수 산정                │
     │    └→ 주3회주: 3회 / 일반주: 2회                  │
     │    └→ 고정 휴무 차감 후 잔여 확정                  │
     │                                                  │
     │  STEP 4. 금·토·일 우선 배분                       │
     │    └→ 합계 누적 최소 직원 우선                     │
     │    └→ 동점: 요일 개별 최소 → 랜덤                 │
     │    └→ 최소 출근 인원 제약 적용                     │
     │                                                  │
     │  STEP 5. 연속 출근 제약 확인 (Hard)               │
     │    └→ 6일 초과 탐지 → swap 시도                   │
     │    └→ swap 불가 → ⚠️ 플래그                      │
     │                                                  │
     │  STEP 6. 평일 잔여 휴무 랜덤 배분                 │
     │    └→ 월~목에 균등 확률 배정                      │
     │    └→ 최소 출근 인원 제약 적용                     │
     │                                                  │
     │  STEP 7. 결과 검증 및 자동 보정                   │
     │    └→ ① 총 휴무 횟수 일치 확인                    │
     │    └→ ② 금토일 균등도 검증 (차이 ≤1 목표)        │
     │    └→ ③ 연속 출근 위반 최종 확인                  │
     │    └→ 실패 시 STEP 4~6 재시도 (최대 5회)          │
     └──────────────────────────────────────────────────┘
       │
       ▼
     {생성 결과}
       │
       ├─ ✅ 모든 검증 통과
       │     │
       │     ▼
       │   [4주 캘린더 뷰(F-09)로 자동 이동]
       │     - 상단: 균등도 점수 ✅ + 경고 0건
       │     - 그리드: 색상 코딩된 28일 스케줄
       │     - 하단: 금토일 균등 현황판(F-10)
       │
       ├─ ⚠️ 부분 경고 (금토일 차이 2 이상 또는 연속 출근 위반)
       │     │
       │     ▼
       │   [캘린더 뷰 + 경고 배너]
       │     - '균등 미달 ⚠️ — 수동 수정 또는 재생성을 권장합니다'
       │     - 위반 셀 주황색 테두리 강조
       │     - [수동 수정] / [재생성] 버튼 강조
       │
       └─ ❌ 점장/부점장 고정 휴무 충돌
             │
             ▼
           [에러 메시지] + F-03 승인 화면으로 이동 유도
```

### 2.6 스케줄 수동 수정 (F-07)

```
[4주 캘린더 뷰] (관리자 모드)
  │
  ▼
[셀 클릭]
  │
  ├─ 일반 셀 (OFF_RANDOM / WORK)
  │     │
  │     ▼
  │   [토글] 출근 ↔ 휴무
  │     │
  │     ▼
  │   {실시간 제약 검증}
  │     ├─ 주별 총 휴무 → 위반 시 빨간 테두리
  │     ├─ 연속 출근 6일 초과 → 주황색 강조
  │     ├─ 최소 출근 인원 → 열 배경 빨간색
  │     └─ 금토일 균등도 → 현황판 실시간 갱신
  │
  ├─ 고정 휴무 셀 (OFF_FIXED)
  │     └→ 🔒 잠금 — 클릭 시 안내: 'F-03 승인 화면에서만 변경 가능'
  │
  └─ 점장/부점장 매장 휴무일 셀 (WORK_LOCKED)
        └→ 🔒 잠금 — 클릭 시 사유 툴팁 표시
  
  │
  ▼
[저장]
  │
  ├─ 제약 위반 없음 → [저장] ✅
  │
  └─ 제약 위반 있음
       │
       ▼
     [강제 저장] 버튼
       │
       ▼
     [확인 팝업] '제약 조건 위반이 있습니다. 그래도 저장하시겠습니까?'
       │
       ├─ [확인] → 저장 (override) ✅
       └─ [취소] → 수정 계속

  ※ [되돌리기(Undo)] → 최대 20단계 이전으로 복원 가능
```

### 2.7 재생성 (F-08)

```
[캘린더 뷰 상단]
  │
  ▼
[🔀 재생성] 버튼 클릭
  │
  ▼
[확인 팝업] '현재 스케줄이 초기화됩니다. 계속하시겠습니까?'
  │
  ├─ [확인]
  │     │
  │     ▼
  │   고정 휴무(OFF_FIXED) 유지 + STEP 4~6 새로 랜덤화
```

## 3. 직원 플로우

### 3.1 로그인 및 홈

```
[앱 접속]
  │
  ▼
[직원 로그인]
  │  - PIN 입력 또는 공유 링크(URL)로 접속
  │
  ▼
[직원 홈]
  │  - 현재 4주 내 스케줄 요약 (다음 출근일, 총 휴무일 수)
  │  - 고정 휴무 신청 상태 (대기/승인/거절)
  │
  ├→ [고정 휴무 신청]
  └→ [내 스케줄 보기]
```

### 3.2 고정 휴무 신청 (F-03)

```
[고정 휴무 신청 화면]
  │
  ▼
{신청 기한 확인} — 4주 시작일 3일 전까지
  │
  ├─ 기한 초과 → ❌ 신청 버튼 비활성화 '신청 기한이 지났습니다'
  │
  └─ 기한 내
       │
       ▼
     [4주 캘린더 표시]
       │  - 날짜 클릭 → 선택 표시
       │
       ▼
     {최대 신청 횟수 확인} — 4주당 최대 2일
       │
       ├─ 초과 → ❌ '이번 4주 주기 신청 한도(2일)를 초과했습니다'
       │
       └─ 한도 내
            │
            ▼
          [고정 휴무 신청] 버튼
            │
            ▼
          상태: '대기 중(Pending)' 표시
            │
            ▼
          {관리자 처리 대기}
            │
            ├─ 승인(Approved) → 스케줄 생성 시 우선 배정 ✅
            └─ 거절(Rejected) → 알림 수신 + 사유 확인

  ※ 대기 중인 신청은 직원이 직접 취소 가능
```

### 3.3 스케줄 조회 (F-09/10)

```
[내 스케줄 보기]
  │
  ▼
[4주 캘린더 뷰 — 조회 전용]
  │  - 자신의 행만 표시 (타 직원 이름 마스킹 또는 숨김)
  │  - 셀 색상 코딩:
  │     흰색     = 출근
  │     파란색    = 일반 휴무
  │     진파란색  = 매장 지정 휴무일 휴무
  │     보라색    = 고정 휴무 (본인 신청)
  │
  ▼
[금토일 균등 현황판]
  │  - 본인의 금/토/일 개별 횟수 + 합계
  │  - 전체 균등도 점수 확인
  │
  ▼
[모바일]
  │  - 1주차씩 스와이프 전환
```

## 4. 핵심 시나리오별 플로우

### 시나리오 A: 첫 4주 스케줄 생성 (처음부터 끝까지)

```
관리자 로그인
    │
    ▼
① [직원 관리] → 전체 직원 등록 (이름, 직급)
    │
    ▼
② [4주 기간 설정] → 시작일 선택 → 매장 지정 휴무일 지정 → 저장
    │
    ▼
③ (직원들이 고정 휴무 신청 — 시작일 3일 전까지)
    │
    ▼
④ [고정 휴무 승인] → 신청 목록 확인 → 승인/거절
    │
    ▼
⑤ [자동 스케줄 생성] → 7단계 알고리즘 실행
    │
    ▼
⑥ [결과 확인] → 캘린더 뷰 + 균등 현황판
    │
    ├─ 만족 → ⑧로 이동
    │
    └─ 불만족
         ├→ [수동 수정] → 셀 편집 + 실시간 검증
         └→ [재생성] → 새로운 랜덤 결과
    │
    ▼
⑦ (반복: 수정/재생성 → 결과 확인)
    │
    ▼
⑧ [스케줄 확정] → 잠금
    │
    ▼
⑨ [공유] → 직원에게 링크/알림 전송
    │
    ▼
⑩ [내보내기] → 엑셀/PDF 다운로드 (인쇄용)
```

### 시나리오 B: 다음 4주 Rollover (Phase 3)

```
관리자 로그인
    │
    ▼
① [4주 기간 설정] → 새 시작일 선택
    │
    ▼
② [템플릿 적용] → 이전 4주 패턴 기반 초안 자동 생성
    │
    ▼
③ (직원 변동 자동 처리)
    │
    ▼
④ 제약 검증 → 필요시 수동 수정
    │
    ▼
⑤ 확정 → 공유
```

### 시나리오 C: 직원의 일반적 사용

```
직원 로그인 (PIN/링크)
    │
    ▼
① [홈] → 다음 출근일/잔여 휴무 확인
    │
    ├→ 4주 시작 전:
    │     [고정 휴무 신청] → 날짜 선택 (최대 2일) → 대기
    │
    └→ 스케줄 확정 후:
          [내 스케줄 보기] → 4주 캘린더 뷰 조회
```
